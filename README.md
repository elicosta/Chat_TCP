# Chat_TCP

Comunicação entre *cliente* e *Servidor* com Socket TCP em **Python 3**.

## Cliente

### Requisitos

* Python 3
* Banco de Dados (De sua preferência)
  * Postgres
  * MariaDB
  * mySQL
  * `python3 servidor.py`

### Utilização

Após servidor está ativado é possivel executar por meio do terminal o cógigo do `servidor.py`

## Servidor

### Requisitos

* Python 3
* Banco de Dados (De sua preferência)
  * Postgres
  * MariaDB
  * mySQL

### Execução

`python3 servidor.py`
